### ✦ &nbsp; Selected Work

**RAG Chatbot · Internal AI Assistant (500+ employees)**
Built the UI for an internal assistant using FastAPI, Pinecone, and LangChain. Delivered fast retrieval and high relevance scores; adopted across the company.

**Back Office API Refactor**
Split a monolithic API into operator and game endpoints. Migrated payloads from XML to Protobuf, reducing payload size and load times. Shipped with zero defects.

**SonarQube CI/CD Integration**
Proposed and implemented SonarQube in the CI/CD pipeline; adopted as a company-wide standard. Raised unit test coverage from ~35% to 78% using JUnit.

**FanDuel Operator Feature · US NJ Compliance**
Implemented feature-flagged free-rounds audit and compliance logic scoped to US NJ production. Designed to be reversible and reusable via Ansible configuration.

**Java 8 → 17 + Spring Boot 3 Migration**
Contributed to an org-wide migration driven by Oracle's 2023 licensing changes. Authored Confluence documentation to onboard future engineers.
